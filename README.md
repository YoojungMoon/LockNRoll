# LockNRoll
저마찰 환경에서의 휠체어 바퀴 미끄럼 방지 보조장치

* Overview
본 프로젝트는 눈길, 빗길, 결빙된 노면 등 저마찰 환경에서 발생하는 휠체어 바퀴 미끄럼 문제를 해결하기 위한 전개형 미끄럼 방지 휠 구조를 설계·제작하는 것을 목표로 한다.
기존 미끄럼 방지 장치가 가지는 상시 저항, 탈부착의 번거로움, 사용자 조작 부담 등의 한계를 개선하고자, 필요할 때만 작동하는 구조와 적은 힘으로 조작 가능한 메커니즘에 초점을 맞추었다.

* Motivation
저마찰 환경에서 휠체어 사용자는 추진력 상실, 방향 제어 어려움, 전복 위험 등에 노출된다.
기존의 체인형 또는 외부 부착형 장치는 사용 전·후 장착이 필요하고, 평상 주행 시 불필요한 저항과 무게 증가를 유발하며, 주행 중 즉각적인 대응이 어렵다는 한계가 있다.
본 프로젝트는 이러한 문제를 해결하기 위해 스터드 타이어의 개념을 차용하되, 휠체어 환경에 맞게 수동 전개형 구조로 재해석하였다.

* Key Idea
- 볼펜형 on/off 메커니즘을 적용하여 돌기 전개 제어
- 레버 구조를 통해 적은 힘으로 작동 가능
- 일괄 돌출 방식의 한계를 극복하기 위해 지면과 비접촉 영역부터 순차적으로 전개되는 구조 채택
